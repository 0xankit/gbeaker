# `beaker task`

Managing tasks for the project

Arguments:

* `--help`: Print help information

* `--version`: Print version information

## Subcommands

### `beaker task new`

Create a new task

Arguments:

* `--help`: Print help information

* `--version`: Print version information

* ` <task>`Name of the task

---

### `beaker task run`

Run a task

Arguments:

* `--help`: Print help information

* `--version`: Print version information

* ` <script>`Name of the task

* ` <args>`
